# 🎵 Music Compass

> Herramienta visual e interactiva para explorar armonía, escalas y acordes en tu instrumento.  
> Basado en **Armonía Ilustrada** de Brian Callipari.

**[▶ Abrir la app](https://tuusuario.github.io/music-compass)**

---

## ¿Qué es?

Music Compass te ayuda a entender cómo funciona la armonía y a aplicarla directamente sobre tu instrumento. Está organizada alrededor de los conceptos del libro *Armonía Ilustrada* de Brian Callipari: funciones tonales, progresiones, dominantes, modulaciones y más.

No requiere instalación. Abrís el link y listo.

## Cómo usar la app

### 1. Elegí tu instrumento
Los botones en la parte superior cambian el instrumento. El mástil o teclado se actualiza automáticamente.

### 2. Elegí la tónica
Las flechas **◀ ▶** navegan por el círculo de quintas. También podés tocar directamente cualquier nota en la pestaña ⭕ Círculo.

### 3. Elegí el modo / escala
El menú desplegable tiene los 7 modos griegos:

| Modo | Carácter |
|------|----------|
| Mayor (Jónico) | Feliz, brillante, estable |
| Dórico | Melancólico pero optimista. Funk, jazz |
| Frigio | Oscuro, misterioso. Flamenco, metal |
| Lidio | Soñador, etéreo. Jazz, bandas sonoras |
| Mixolidio | Bluesy, dominante. Blues, rock, funk |
| Menor (Eólico) | Triste, serio. Escala menor natural |
| Locrio | Muy inestable. Para tensiones especiales |

## Pestañas

### 🎼 Armonía
La sección principal. Tiene **9 conceptos del libro** con explicación y acordes interactivos. Al tocar un acorde, se resalta en el instrumento.

| # | Concepto | Descripción |
|---|----------|-------------|
| 1 | **Funciones Tonales** | Los 7 acordes de la tonalidad agrupados por función |
| 2 | **Progresión 2–5–1** | El motor armónico del jazz: ii → V7 → I |
| 3 | **Progresión 6–2–5–1** | Extensión del 2–5–1, muy común en standards |
| 4 | **Cadencias** | Perfecta (V7→I), plagal (IV→I) y alternativas |
| 5 | **Dominantes Secundarios** | El V7 de cada acorde de la tonalidad |
| 6 | **Cadena de Dominantes** | Dominantes separados por cuartas |
| 7 | **Sustituto Tritonal** | Reemplazar el V7 por el acorde 6 semitonos arriba |
| 8 | **Puente / Modulación** | Moverse entre tonalidades por acorde pivote |
| 9 | **Mayor ↔ Menor (Préstamo)** | Acordes prestados de la tonalidad paralela menor |

### 🎸 Instrumento
Muestra la escala o el acorde activo sobre el mástil o teclado. Usá los botones **◀ Anterior / Siguiente ▶** para navegar por los trastes.

### ⭕ Círculo de Quintas
Círculo interactivo. Tocá cualquier nota para cambiar la tónica. Las notas de la escala activa se muestran en verde.

## Ejemplo práctico

Querés componer en **La menor (Am)**:

1. Tónica → **A** con las flechas ◀ ▶
2. Modo → **Menor (Eólico)**
3. Pestaña **🎼 Armonía** → *Funciones Tonales* para ver los 7 acordes disponibles
4. Tocá cada acorde para ver sus notas en el instrumento
5. Seleccioná *Progresión 6–2–5–1* para armar la estructura: **F – B° – E7 – Am**
6. Para el puente, usá *Puente / Modulación* para llegar a una tonalidad vecina

## Glosario

| Término | Definición |
|---------|-----------|
| **Tónica** | La nota raíz que define la tonalidad |
| **Modo** | Escala con carácter específico según el orden de sus intervalos |
| **Intervalo** | Distancia en semitonos entre dos notas |
| **Tríada** | Acorde de 3 notas: raíz, 3ra y 5ta |
| **Dominante** | El acorde de máxima tensión que quiere resolver al I |
| **Cadencia** | Progresión que cierra o resuelve una frase musical |
| **Tritono** | Distancia de 6 semitonos. La más tensa posible |
| **Sustituto Tritonal** | Dominante que reemplaza a otro por compartir el mismo tritono |
| **Modulación** | Cambiar de tonalidad dentro de una pieza musical |
| **Acorde Pivote** | Acorde que pertenece a dos tonalidades, sirve de puente entre ellas |
| **Préstamo Modal** | Usar acordes de la tonalidad paralela menor dentro de una tonalidad mayor |
| **Cadena de Dominantes** | Sucesión de dominantes por cuartas, cada uno jalando al siguiente |
